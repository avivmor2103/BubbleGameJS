- To add a timer you need to enter a time in the white rectangle and click the Start timer button.

- if you choose to leave the page the browser will give a message.
- When the last disc remains, a message will be sent to the user that this disc has "won" according to its color.
-In case that the time runs out in the timer, and in addition no disk remains last,
 a message will be sent to the user.
